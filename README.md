# HurahiOS

## Compilation
* Run command `make` in the directory of the Makefile
* To use all available cores, use 
```bash
make -j`nproc`
```
* Clean useing `make clean`
* Specify arch with `ARCH=x make`

## How to run
* Run the OS using qemu (https://www.qemu.org/)
